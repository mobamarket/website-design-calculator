== Description ==
Custom Cost Calculator is a powerful, user-friendly plugin for creating interactive price estimation and quotation forms with a drag-and-drop builder. Perfect for service businesses, e-commerce, and lead generation.

= Features =
* Drag-and-drop form builder (React/Vue-based)
* Unlimited calculators and forms
* Input types: text, quantity, dropdown, toggle, radio, checkbox, range slider, file upload, date/time picker, and more
* Conditional logic for dynamic forms
* Custom formula builder for advanced calculations
* WooCommerce integration for dynamic product pricing
* Payment gateways: PayPal, Stripe, Razorpay, Cash on Delivery
* 20+ pre-built templates for various industries
* Order management dashboard
* Appearance customization (colors, fonts, layouts, CSS)
* Shortcode and page builder integration (Elementor, WPBakery, Gutenberg)
* Lead generation and email integration (Mailchimp, SendGrid)
* Multilingual support (WPML, Polylang)
* Secure, optimized, and extensible

= Premium Features =
* Advanced elements (multi-range slider, image dropdown, etc.)
* More integrations and templates
* Priority support

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/custom-cost-calculator` directory, or install via the WordPress plugins screen.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to 'Cost Calculators' in the WordPress admin menu to create your first calculator.

== Frequently Asked Questions ==
= Can I use this with WooCommerce? =
Yes! You can embed calculators on product pages and use calculated prices in the cart.

Yes, advanced features are available in the premium version.
